# I.操作步骤

- 在开始程序里搜索cmd,打开cmd.exe
- 切换到E盘的方法 e:然后按回车键
- 切换到工程目录的方法，输入

<pre>
cd projects
cd LisarWang.github.io
</pre>

- 创建一篇文章：

<pre>rake post title="文章名"</pre>

# II.预览文章

- jekyll serve：在编辑器里输入，再按回车

- http://localhost:400

# III.提交修改

- git stastus
- git add-A
- git commit-m"zhushi"
- git push origin master

# IV.在浏览器中打开

https://lisarwang.github.io
